## 여러 이미지를 하나의 PDF로 합치기

회의록, 계약서 스캔본, 포트폴리오 등 여러 이미지를 하나의 PDF로 묶어야 할 때가 많습니다. 이미지 PDF 변환기를 사용하면 복잡한 소프트웨어 없이 브라우저에서 바로 처리할 수 있습니다.

## 단계별 가이드

### 1단계: 이미지 업로드

'이미지를 클릭하거나 드래그하여 업로드' 영역에 여러 이미지를 한 번에 선택하거나 드래그&드롭합니다. 파일 선택 창에서 Ctrl(Mac은 Cmd)을 누르고 클릭하면 여러 파일을 동시에 선택할 수 있습니다.

### 2단계: 순서 확인

업로드된 이미지 목록을 확인합니다. 이미지는 업로드된 순서대로 PDF 페이지가 됩니다. 순서가 잘못된 경우 X 버튼으로 해당 이미지를 삭제하고 올바른 순서로 다시 업로드하세요.

### 3단계: PDF 옵션 설정

**페이지 방향**:
- 자동: 이미지의 가로/세로 비율에 따라 자동 결정
- 세로(Portrait): 모든 페이지를 세로 방향으로 통일
- 가로(Landscape): 모든 페이지를 가로 방향으로 통일

**여백**:
- 여백 없음: 이미지가 페이지 전체를 채움
- 작은 여백(10mm): 가장 많이 사용되는 설정
- 표준 여백(20mm): 공문서 형태에 적합

### 4단계: 변환 및 다운로드

'PDF로 변환하기' 버튼을 클릭하면 진행 상황이 표시되며 변환이 완료되면 자동으로 PDF 파일이 다운로드됩니다.

## 품질 유지 팁

- **원본 해상도 유지**: 이미지 PDF 변환기는 이미지 품질을 최대한 유지합니다
- **적절한 이미지 크기**: 너무 큰 이미지(50MB 이상)는 변환 속도가 느려질 수 있습니다
- **인쇄 품질**: 인쇄용 PDF가 필요하다면 300DPI 이상의 원본 이미지를 사용하세요

## 활용 사례별 설정 추천

| 용도 | 페이지 방향 | 여백 |
|------|-------------|------|
| 서류 제출 | 자동 | 10mm |
| 사진 포트폴리오 | 자동 | 0mm |
| 공문서 스캔 | 세로 | 20mm |
| 파노라마/가로 이미지 | 가로 | 10mm |

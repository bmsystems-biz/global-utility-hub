## WiFi QR코드의 편리함

카페, 사무실, 에어비앤비 등에서 WiFi 비밀번호를 일일이 알려주는 것은 번거롭습니다. WiFi QR코드를 인쇄해두면 손님이 스마트폰으로 스캔하는 것만으로 WiFi에 자동 연결됩니다.

## WiFi QR코드 형식

QR코드에 저장할 텍스트를 아래 형식으로 입력하세요:

```
WIFI:T:WPA;S:네트워크이름;P:비밀번호;;
```

각 항목 설명:
- **T**: 보안 방식 (`WPA`, `WEP`, `nopass`)
- **S**: WiFi 네트워크 이름 (SSID)
- **P**: WiFi 비밀번호

## 예시

- 네트워크 이름: `CafeNetwork`
- 비밀번호: `cafe1234`

```
WIFI:T:WPA;S:CafeNetwork;P:cafe1234;;
```

이 텍스트를 QR코드 생성기에 입력하면 됩니다.

## 주의사항

1. **대소문자 구분**: 네트워크 이름과 비밀번호는 대소문자를 정확히 입력해야 합니다
2. **특수문자**: 비밀번호에 특수문자(`"`, `;`, `,`, `\`)가 포함된 경우 앞에 `\`를 추가해야 합니다
3. **숨겨진 네트워크**: 숨겨진 SSID는 `H:true;`를 추가합니다: `WIFI:T:WPA;S:이름;P:비밀번호;H:true;;`
4. **비밀번호 없는 네트워크**: `WIFI:T:nopass;S:네트워크이름;;`

## 개인정보 보호

글로벌 유틸리티 허브의 QR코드 생성기는 모든 처리가 브라우저에서 이루어집니다. WiFi 비밀번호가 서버로 전송되지 않으므로 안전하게 사용할 수 있습니다.

## 실제 활용 예시

생성한 QR코드를 출력해 카운터나 테이블 위에 놓아두세요. "WiFi 연결은 QR코드를 스캔하세요"라는 안내문과 함께 배치하면 손님들이 직관적으로 이용할 수 있습니다.

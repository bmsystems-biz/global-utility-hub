📱 QR코드 생성기

URL, 텍스트, 연락처 등을 QR코드로 즉시 변환하세요. 완전 무료, 서버 전송 없음, PNG 다운로드 지원합니다.

💡 빠른 예시

예시 1 – URL을 QR코드로
입력
https://util.bmsystems.biz
결과
256×256 QR코드 이미지 생성
용도
명함, 포스터, 간판에 인쇄
예시 2 – WiFi 정보 QR코드
입력
WIFI:T:WPA;S:NetworkName;P:Password;;
결과
WiFi 자동 연결 QR코드
용도
카페, 사무실 입장 시 WiFi 연결 편의 제공

❓ 자주 묻는 질문

QR코드(Quick Response Code)는 2차원 바코드의 일종으로, URL, 텍스트, 연락처 등의 정보를 저장합니다. 스마트폰 카메라로 스캔하면 저장된 정보를 즉시 확인할 수 있습니다.
QR코드가 생성되면 '⬇️ PNG 저장' 버튼을 클릭하면 고화질 PNG 파일로 다운로드됩니다. 모바일에서는 이미지를 길게 눌러 저장할 수도 있습니다.
URL, 일반 텍스트, 이메일(mailto:), 전화번호(tel:), SMS, WiFi 정보(WIFI:T:WPA;S:이름;P:비밀번호;;), 연락처(vCard), 지도 위치 등 다양한 정보를 저장할 수 있습니다.
인쇄용으로는 512px 이상, 디지털 화면용은 256px, 명함 등 소형 인쇄물은 128px를 권장합니다. 크기가 클수록 스캔 인식률이 높아집니다.
아니요. 모든 처리가 브라우저(클라이언트)에서만 이루어집니다. 입력한 데이터는 절대 서버로 전송되지 않아 개인정보가 안전하게 보호됩니다.
오류 수정 수준은 QR코드가 손상되었을 때 복원 가능한 정도를 나타냅니다. L(7%), M(15%), H(30%) 순으로 내구성이 높아지지만 저장 용량이 줄어듭니다. 일반적으로 M 수준을 권장합니다.

JPG, PNG, WebP 형식으로 이미지를 변환합니다. 파일이 서버로 전송되지 않습니다.

🖼️
이미지를 끌어다 놓거나 클릭하여 선택
JPG, PNG, WebP, GIF, BMP 지원

❓ 자주 묻는 질문

JPG는 손실 압축 방식으로 파일 크기가 작지만 품질이 약간 저하됩니다. 사진에 적합합니다. PNG는 무손실 압축으로 투명 배경(알파 채널)을 지원합니다. 로고, 아이콘, 스크린샷에 적합합니다.
Google이 개발한 차세대 이미지 형식으로, JPG보다 25~35% 작은 파일 크기에 비슷한 품질을 제공합니다. 투명도도 지원합니다. 최신 브라우저에서는 모두 지원하므로 웹 최적화에 권장됩니다.
아닙니다. 모든 변환은 브라우저 내부에서 처리됩니다. 이미지가 외부 서버로 전송되지 않으므로 개인 정보와 보안이 보장됩니다.
JPG는 투명도를 지원하지 않습니다. PNG의 투명 영역은 흰색 배경으로 채워집니다. 투명 배경이 필요하다면 PNG나 WebP 형식으로 유지하세요.
JPG와 WebP의 손실 압축 수준을 설정합니다. 100%는 최고 품질(큰 파일), 낮을수록 파일이 작아지지만 화질이 저하됩니다. 일반적으로 80~90%가 파일 크기와 품질의 균형이 좋습니다. PNG는 무손실이라 품질 설정이 적용되지 않습니다.

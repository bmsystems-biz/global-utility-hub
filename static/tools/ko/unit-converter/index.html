❓ 자주 묻는 질문

1인치 = 2.54cm 입니다. 따라서 6피트(72인치) = 182.88cm입니다.
1파운드(lb) = 약 453.6g = 0.4536kg입니다. 반대로 1kg = 약 2.205lb입니다.
℃ = (℉ - 32) × 5/9 입니다. 예) 98.6℉ = (98.6-32)×5/9 = 37℃ (정상 체온)
1평 = 3.30579m² ≈ 3.3m²입니다. 33평 아파트 ≈ 약 109m²
미국 갈런 기준 1갤런 = 약 3.785L입니다. 영국 갤런은 약 4.546L입니다.

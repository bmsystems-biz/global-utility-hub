❓ 자주 묻는 질문

웹용으로는 70~85%가 파일 크기와 화질의 균형이 좋습니다. 인쇄용이나 보관용은 90~95%를 권장합니다. PNG는 무손실 형식이라 품질 슬라이더가 적용되지 않습니다.
WebP는 JPG보다 약 25~35% 더 작은 파일 크기에 비슷한 화질을 제공합니다. 모든 최신 브라우저가 지원하므로 웹 사용에는 WebP를 권장합니다. 범용 호환성이 필요하면 JPG를 선택하세요.
아닙니다. 모든 압축은 브라우저 내 Canvas API를 사용해 처리됩니다. 파일이 외부 서버로 전송되지 않아 개인정보가 완전히 보호됩니다.
JPG는 투명도를 지원하지 않아 투명 영역이 흰 배경으로 채워집니다. 투명도를 유지하려면 PNG나 WebP 형식을 선택하세요.

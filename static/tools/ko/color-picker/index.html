?? 색상 선택기

색상을 선택하고 HEX, RGB, HSL 형식으로 즉시 변환합니다.

프리셋 색상

? 자주 묻는 질문

#RRGGBB 형식으로 빨강(Red), 초록(Green), 파랑(Blue) 각 채널을 16진수 2자리로 표현합니다. 예를 들어 #FF0000은 순수한 빨강, #000000은 검정, #FFFFFF는 흰색입니다. CSS와 웹 디자인에서 가장 널리 사용됩니다.
동일한 색상을 다르게 표현하는 방식입니다. RGB는 rgb(255, 0, 0)처럼 10진수로, HEX는 #FF0000처럼 16진수로 표기합니다. 프로그래밍에서는 수학적 조작이 쉬운 RGB가 유리하고, 디자이너는 주로 HEX를 사용합니다.
HSL은 색조(Hue, 0~360°), 채도(Saturation, 0~100%), 명도(Lightness, 0~100%)로 색상을 표현합니다. 직관적이어서 CSS에서 특정 색상을 밝게/어둡게 조정할 때 RGB보다 편리합니다. 예: hsl(217, 91%, 60%)
HEX에서는 8자리 형식 #RRGGBBAA를 사용합니다 (예: #3b82f680). RGB에서는 rgba(59, 130, 246, 0.5), HSL에서는 hsla(217, 91%, 60%, 0.5)처럼 a(alpha) 값을 0~1 사이로 추가합니다.
CSS에는 140개의 표준 색상 이름이 있습니다 (red, blue, tomato, cornflowerblue 등). 이 중 16개는 기본 색상(Basic Color Keywords)이고, 나머지는 확장 색상 이름입니다. 브라우저마다 렌더링이 약간 다를 수 있어 실무에서는 HEX 사용을 권장합니다.

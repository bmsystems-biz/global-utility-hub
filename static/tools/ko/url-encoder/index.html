🔗 URL 인코더

텍스트를 URL 퍼센트 인코딩으로 변환하거나 디코딩합니다.

❓ 자주 묻는 질문

URL에서 특수문자나 한글 등 비ASCII 문자는 %XX 형식(퍼센트 인코딩)으로 변환해야 합니다. 예: 안녕 → %EC%95%88%EB%85%95, 공백 → %20
encodeURI는 URL 전체를 인코딩하므로 /, ?, # 등 URL 구조 문자는 유지합니다. encodeURIComponent는 쿼리 파라미터 등 URL 일부를 인코딩할 때 사용하며 특수문자를 더 철저히 변환합니다. 이 도구는 encodeURIComponent를 사용합니다.
HTML form 인코딩(application/x-www-form-urlencoded)에서는 공백이 +로 인코딩됩니다. 퍼센트 인코딩에서는 공백이 %20입니다. 이 도구는 %20을 사용합니다.

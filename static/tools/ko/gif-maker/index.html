❓ 자주 묻는 질문

각 이미지(프레임)가 표시되는 시간(밀리초)입니다. 200ms면 초당 5프레임, 100ms면 초당 10프레임입니다. 자연스러운 애니메이션은 80~150ms를 권장합니다.
파일을 선택한 순서대로 프레임이 추가됩니다. 파일명을 숫자로 정렬(예: 001.jpg, 002.jpg)해두면 원하는 순서로 선택하기 쉽습니다.
아닙니다. gif.js 라이브러리를 이용해 모든 처리가 브라우저 내에서 이루어집니다.
출력 크기(너비×높이)를 줄이거나 프레임 수를 줄이세요. GIF는 색상을 256색으로 제한하는 특성상 사진보다 단순한 그래픽에서 더 작은 파일이 생성됩니다.

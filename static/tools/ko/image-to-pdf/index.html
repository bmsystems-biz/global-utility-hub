📄 이미지 PDF 변환기

JPG, PNG, WebP 등 이미지를 PDF로 무료 변환합니다. 서버에 파일이 전송되지 않아 개인정보가 완전히 보호됩니다.

🖼️
이미지를 클릭하거나 드래그하여 업로드
JPG, PNG, GIF, WebP, BMP 지원 · 여러 파일 선택 가능
0개 이미지
    🔒
    개인정보 보호

    이미지는 서버에 전송되지 않습니다. 모든 처리가 브라우저에서만 이루어집니다.

    📐 사용 방법

    • 이미지를 드래그&드롭하거나 클릭하여 업로드합니다.
    • 여러 이미지를 선택하면 목록의 순서대로 PDF 페이지가 생성됩니다.
    • 페이지 방향과 여백을 설정합니다.
    • 'PDF로 변환하기' 버튼을 클릭하면 자동으로 PDF 파일이 다운로드됩니다.

    💡 활용 예시

    예시 1 – 스캔 이미지를 PDF로
    입력
    영수증, 계약서 JPG 사진 3장
    결과
    3페이지 PDF 파일 1개 생성
    용도
    이메일 제출, 클라우드 저장
    예시 2 – 포트폴리오 PDF 제작
    입력
    작품 이미지 PNG 10장
    결과
    10페이지 포트폴리오 PDF
    용도
    취업 지원 포트폴리오 제출

    ❓ 자주 묻는 질문

    JPG(JPEG), PNG, GIF, WebP, BMP 등 브라우저에서 표시 가능한 모든 이미지 형식을 지원합니다.
    아니요. 모든 처리가 브라우저에서만 이루어집니다. 이미지는 절대 서버로 전송되지 않아 개인정보가 100% 보호됩니다.
    네, 여러 이미지를 업로드하면 목록 순서대로 하나의 PDF에 한 페이지씩 배치됩니다.
    A4 용지(210×297mm) 기준으로 이미지가 비율을 유지하며 삽입됩니다. 페이지 방향(가로/세로)은 이미지 비율에 따라 자동 선택하거나 직접 설정할 수 있습니다.
    이론상 제한은 없지만, 브라우저 메모리 내에서 처리하므로 매우 큰 이미지(50MB 이상)는 성능에 영향을 줄 수 있습니다. 파일당 10MB 이하를 권장합니다.

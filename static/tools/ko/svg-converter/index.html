❓ 자주 묻는 질문

SVG(Scalable Vector Graphics)는 XML 기반의 벡터 이미지 형식입니다. 어떤 크기로 확대해도 품질이 유지됩니다. 로고, 아이콘, 일러스트에 주로 사용됩니다.
일부 소프트웨어(Word, SNS 등)나 오래된 브라우저는 SVG를 지원하지 않습니다. PNG/JPG로 변환하면 어디서든 사용할 수 있습니다. 단, 래스터 이미지는 SVG처럼 무한 확대가 불가능합니다.
아닙니다. 모든 변환은 브라우저 내 Canvas API를 사용해 처리됩니다.
SVG는 벡터이므로 원하는 크기로 선명하게 출력됩니다. 인쇄용이라면 1200px 이상, 웹용 아이콘이라면 64~256px가 적합합니다.

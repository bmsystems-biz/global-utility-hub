❓ FAQ

SVG (Scalable Vector Graphics) is an XML-based vector image format. Unlike raster formats (JPG, PNG), SVG maintains crisp quality at any size. It's commonly used for logos, icons, and illustrations.
Some applications (Word documents, older browsers, social media platforms) don't support SVG. Converting to PNG/JPG ensures universal compatibility. Note that raster images can't be scaled infinitely like SVG.

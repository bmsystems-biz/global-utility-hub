For web use, 70-85% provides a good balance between file size and visual quality. Use 90-95% for print or archival purposes. PNG is lossless so the quality slider does not apply to it.
WebP provides 25-35% smaller file sizes than JPG at comparable quality. All modern browsers support it, making it the recommended choice for web use. Choose JPG for maximum compatibility with older software.
